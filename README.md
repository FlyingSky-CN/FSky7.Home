# FSky7.Home

**FlyingSky's Website - HomePage**

一个简约的个人网站主页，基于 Vue 和 Vuetify 开发。

如果你在寻找基于 MDUI 编写的版本，请前往 [MDUI branch](https://github.com/FlyingSky-CN/FSky7.Home/tree/mdui)。
